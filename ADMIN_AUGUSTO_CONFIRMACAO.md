## Tipos de Usuários Disponíveis

Ao gerenciar usuários, você pode atribuir 3 tipos de papéis:

### Admin (Administrador)
- Acesso total
- Pode gerenciar usuários
- Pode modificar tudo
- **Use com moderação** - atribua apenas a pessoas de confiança

### User (Usuário)
- Acesso completo às ferramentas de análise
- Pode criar e editar análises
- Pode gerar relatórios
- **Ideal para engenheiros e analistas**

### Viewer (Visualizador)
- Acesso apenas para visualização
- Não pode modificar dados
- Pode ver relatórios
- **Ideal para stakeholders e clientes**

## Segurança e Boas Práticas

### Recomendações Importantes:

1. **Use Senha Forte**
   - Como admin, sua conta tem acesso total
   - Use senha complexa e única
   - Não compartilhe suas credenciais

2. **Gerencie Admins com Cuidado**
   - Limite o número de administradores
   - Atribua papel de admin apenas quando necessário
   - Revise periodicamente quem tem acesso admin

3. **Bloqueie Contas Inativas**
   - Usuários que não usam mais o sistema devem ser bloqueados
   - Isso aumenta a segurança

4. **Monitore Atividades**
   - Verifique último acesso dos usuários
   - Revise periodicamente a lista de usuários
   - Mantenha apenas usuários ativos

5. **Proteções Automáticas**
   - Você não pode bloquear sua própria conta
   - Você não pode excluir sua própria conta
   - Você não pode alterar seu próprio papel
   - Essas proteções evitam acidentes
